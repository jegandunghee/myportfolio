# Personal Portfolio

React(CRA) 기반의 개인 포트폴리오입니다.  
상황과 상태 변화에 따라 UI가 전환되는 인터랙션 구조를 중심으로 구성했습니다.

## Overview
반응형 레이아웃과 UI 인터랙션을 포함한 개인 웹 포트폴리오입니다.  
컴포넌트 구조화와 유지보수를 고려한 설계를 목표로 했습니다.

## Features
- 상태 변화에 따른 UI 전환
- 반응형 레이아웃 (Desktop / Tablet / Mobile)
- 프로젝트 목록 및 상세 (배포 링크 / GitHub 레포 연결)
- About / Skills / Contact 섹션 구성
- GSAP 기반 UI 인터랙션

## Tech Stack
**Frontend**  
- React (CRA), JavaScript (ES6+)

**Styling**  
- SCSS, CSS Modules

**Routing**  
- React Router DOM

**Interaction**  
- GSAP

## Project Structure
npm install
npm start

## Build & Deploy
npm run build

## Notes

컴포넌트 구조는 재사용성과 유지보수를 고려해 분리했습니다.

불필요한 라이브러리 사용을 지양하고, 필요한 기능 위주로 구성했습니다.

각 프로젝트에는 구현 의도와 주요 로직을 함께 정리했습니다.
